## How to Run
1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/Simple-Notepad-Swing.git
   ```
2. Navigate to the project folder:
   ```sh
   cd Simple-Notepad-Swing
   ```
3. Compile and run the application:
   ```sh
   javac SimpleNotepad.java
   java SimpleNotepad
   ```
